🌟 المشروع: AI-Based Deforestation Risk Prediction – Amazon Basin

الفكرة الرئيسية

نتوقع المناطق المعرضة لإزالة الغابات باستخدام Spatial Analysis + AI.
The model predicts deforestation risk using spatial and environmental variables.

مراحل المشروع (الترتيب الحقيقي للشغل)

Data ➔ Risk Model ➔ Visualization ➔ Spatial Analysis ➔ Model Validation

1️⃣ المرحلة الأولى — البيانات

الملف: forest_data_clean.json | 150,000 نقطة
كل نقطة فيها: (موقع + درجة خطر) lat, lon, risk_score, risk_norm, elevation, color

2️⃣ المرحلة الثانية — حساب الخطر (Risk Model)

الموديل يعمل حسب Risk Score. كلما زادت العوامل أدناه، يزيد خطر إزالة الغابات:

Risk Index =
0.30 Forest Loss + 0.20 Distance to Roads
+ 0.15 Population Density + 0.15 Fire History
+ 0.10 Slope + 0.10 Elevation

3️⃣ المرحلة الثالثة — الخريطة التفاعلية

  • 3D Risk Map: أعمدة تمثل ارتفاع العمود = risk_score
  • Heatmap: توضح مناطق الخطر العالية.
  • Hotspots: تحدد أهم تجمعات الخطر.
  • Pie Chart: يبين توزيع الخطر (مثال: High Risk ≈ 64%, Medium ≈ 22%, Low ≈ 14%)

4️⃣ المرحلة الرابعة — التحليل

الكود analysis.py يعمل الآتي:
1. ROC Curve: يقيس هل الموديل صحيح؟ (النتيجة: AUC ≈ 0.82 وده ممتاز)
2. DBSCAN: يحدد Hotspots (تجمعات إزالة الغابات)
3. Fire Validation: يقارن بين المناطق الخطرة والحرائق للتحقق المستقل.

5️⃣ المرحلة الخامسة — التحقق من الموديل (Model Validation)

المقياس: AUC Score
النتيجة: AUC = 0.82

التفسير:
0.5 = ضعيف | 0.7 = جيد | 0.8 = قوي (موديلك قوي جداً).

The model achieved an AUC score of 0.82, indicating strong predictive performance.

ANALYSIS & VALIDATION

The model reached an AUC score of 0.82, indicating strong predictive performance. Spatial validation was performed by correlating high-risk predictions with active fire events and historical hotspots.

Hotspot Clusters: Identified using DBSCAN clustering to correlate fire intensity with predicted risk zones.

DEEP LEARNING & MATHEMATICAL MODEL

Semantic Segmentation

The core engine uses a U-Net architecture for binary segmentation. The model was trained using 150,000 spatial samples with multi-temporal features.
