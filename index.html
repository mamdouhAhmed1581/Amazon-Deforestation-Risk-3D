⚙️ STUDY AREA & METHODOLOGY

Focusing on Rondônia, Brazil, this model analyzes spatial-temporal patterns to identify illegal logging threats using our proprietary GeoAI Engine.

Framework Path:
Satellite Imagery Acquisition → Deep Learning Feature Segmentation → Spatiotemporal Risk Surface Projection.

📊 ANALYSIS & VALIDATION

Prediction accuracy was validated against ground-truth datasets. The model reached an AUC score of 0.82, indicating robust predictive performance.

Hotspot Clusters: Specifically identified using DBSCAN clustering algorithms to correlate fire intensity with high-risk degradation zones.

🧠 DEEP LEARNING & MATHEMATICAL MODEL

SEMANTIC SEGMENTATION

The core engine utilizes a U-Net architecture for pixel-wise feature extraction from multi-spectral bands. The output is processed through a spatial risk equation:

Risk Index = 0.30(FL) + 0.20(DR) + 0.15(PD) +
0.15(FH) + 0.10(SL) + 0.10(EL)

* FL: Forest Loss, DR: Road Proximity, PD: Pop Density, FH: Fire History, SL: Slope, EL: Elevation.

Factor Source Weight
Forest Loss PRODES/Sat 30%
Road proximity OpenStreetMap 20%
Population Density GHSL 15%
Active Fires VIIRS / MODIS 15%
Elevation SRTM (NASA) 10%
Slope Digital Terrain 10%
