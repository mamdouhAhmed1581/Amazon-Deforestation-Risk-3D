🛰️ Satellite Data Acquisition

The platform integrates multi-temporal satellite imagery from Landsat 8 and Sentinel-2 sensors. Data preprocessing includes atmospheric correction, cloud masking, and temporal compositing to create analysis-ready surface reflectance products.

  • Landsat 8 OLI/TIRS — 30m resolution, 16-day revisit
  • Sentinel-2 MSI — 10m resolution, 5-day revisit
  • MODIS Terra/Aqua — 250m daily fire detection
  • SRTM DEM — 30m elevation and slope data
  • Hansen Global Forest Change v1.10

🔧 Feature Engineering

Twelve geospatial features are computed from multi-source data. Each feature is normalized to [0, 1] range using min-max scaling. Feature selection is validated using recursive feature elimination (RFE) with cross-validation.

  • Proximity Features: Distance to roads, rivers, urban centers
  • Spectral Indices: NDVI, EVI, NBR vegetation health
  • Topographic: Elevation, slope, aspect from SRTM
  • Historical: Previous forest loss rate (2000–2024)
  • Socioeconomic: Population density, land tenure status
  • Policy: Protected area status, logging concessions

🧠 AI Model Architecture

The Neural-Prophet Engine v2.1 is a hybrid deep learning model combining gradient-boosted decision trees (XGBoost) with a 4-layer feedforward neural network. The ensemble approach leverages the strengths of both methods.

  • Input Layer: 12 features → Dense(64, ReLU)
  • Hidden Layers: Dense(32, ReLU) → Dense(16, ReLU) → Dropout(0.3)
  • Output: Dense(1, Sigmoid) → Risk probability [0, 1]
  • Optimizer: Adam (lr=0.001, weight_decay=1e-5)
  • Loss: Binary cross-entropy with class weights
  • Training: 100 epochs, early stopping (patience=15)
  • Validation: Stratified 5-fold cross-validation

🗺️ Spatial Prediction & Visualization

Risk scores are projected onto a 1km × 1km grid covering the study area. The dashboard uses Deck.GL for WebGL-accelerated 3D column visualization and Mapbox GL JS for basemap rendering.

  • Grid size: ~150,000 cells (1km² resolution)
  • Column height: proportional to risk score
  • Color mapping: Green (low) → Yellow (medium) → Red (high)
  • Interactive tooltips with per-cell risk breakdown
  • Temporal animation across prediction years
